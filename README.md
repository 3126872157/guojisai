## 2024/9/15

位置环&速度环双环pid控制时，位置环不给i，速度环不给d！！！

关节电机因为不同状态受到的阻力不同，所以不同状态时同一组pid参数的控制效果不同，

可以根据不同运动状态设置不同的pid参数
